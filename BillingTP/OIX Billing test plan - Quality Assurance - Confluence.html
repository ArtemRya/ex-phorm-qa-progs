Preconditions

Download Billing.P24.calco.1.2.xls. Rename it to Billing.P24.pass.xls

Open it for editing in Open Office Calc or MS Excel - below steps would instruct you to put some actual values in some cells, check OIX UI values correspond some others and send Ad Server requests stored in some other cells.
Save it after each edition (making backups is a good idea too).

After this test plan completion you'll be asked to attach your  Billing.P24.pass.xls with all your changes to the test plan

attach it even if you hasn't succeed to finish the Test Plan (see #Postconditions)

1.1.2. Generate stats

Don't break Fraud Conditions

 

  1. Open 'Calco' tab. Find rows of your Ad Group (chosen in above test).

  2. Open some other browser for generating stats.
  3. Clear cache & cookies, get persistent UID (how-to-instruction - Central colo is recommended, make sure AdServer is not shut down and is not disconnected from the database).
  4. Generate stats for the 1st tag
    1. find 1st populated intersection of your Ad Group, copy ad request from its brown cell;
    2. in the stats browser open new tab, paste ad request into the address bar and press Enter
      => Creative of the chosen Ad Group should appear => +1 generated impression
    3. [optional] click the impression => chosen Ad Group Creative Click URL us opened in a new window/tab => +1 generated click
    4. [CPA only, optional] click the link to generate an action ("To generate paid action click here")  => page with 1 pixel image is displayed => +1 generated action
